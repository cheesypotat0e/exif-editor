EXIF Date Editor

This page edits existing EXIF date/time fields in-place

It looks for Image DateTime, DateTimeOriginal, DateTimeDigitized, GPSDateStamp, and GPSTimeStamp if present.

And reconstructs the EXIF data back into a new JPEG file.

  • no server
  • no uploads
  • no external libraries
  • no internet required
  • just HTML and JavaScript

Drag & drop a JPEG here, or click to select a file
Only one file at a time. JPEG only.
